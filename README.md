# Единый ФПЦ — лендинг

Премиум-лендинг финансово-правового центра. Чистый HTML + CSS + JS, без фреймворков.
Mobile-first, SEO-разметка (JSON-LD), scroll-анимации.

## Структура
```
efpc-site/
├── index.html        # вся страница
├── css/style.css     # стили (брендовые цвета — в :root сверху)
├── js/main.js        # меню, аккордеон, счётчики, отправка формы
├── assets/           # логотип, favicon
├── package.json
└── vercel.json
```

## ⚠️ Что нужно сделать перед публикацией

### 1. Подключить форму к почте (Яндекс.Облако + Яндекс 360)
Заявки принимает **функция в Яндекс.Облаке** (всё в РФ, 152-ФЗ соблюдён): она шлёт письмо
на **ediniyfpc@efpts.ru** через SMTP Яндекс 360. Код и пошаговая инструкция —
в [`serverless/README.md`](serverless/README.md).

Коротко: развернуть `serverless/handler.py` как функцию, задать переменные
(`SMTP_USER`, `SMTP_PASSWORD` — пароль приложения Яндекс 360, `MAIL_TO`), сделать функцию
публичной и вписать её URL в `js/main.js` (константа `ENDPOINT`).

> Web3Forms/иностранные form-сервисы намеренно НЕ используем: перс. данные россиян
> должны обрабатываться в РФ (152-ФЗ), плюс иностранные API нестабильны из-за РКН.

### 2. Заполнить плейсхолдеры
- **Цены** — сейчас скрыты (кнопка «Рассчитать стоимость»). Если решите показывать — добавим.
- **Документы/лицензии** — блок «Документы и доверие» содержит рамки-заглушки.
  Заменить на реальные сканы СРО / аттестатов / наград (или убрать блок, если пока нет).
- **Политика конфиденциальности** — ссылка в футере ведёт на `#`. Добавить страницу/файл.
- **Кейсы** — взяты со старого сайта ефпц.рф. Можно дополнить новыми.

## Запуск локально
Просто открыть `index.html` в браузере. Или:
```bash
cd efpc-site && python3 -m http.server 8080
# открыть http://localhost:8080
```

## Деплой
**GitHub Pages:**
```bash
cd efpc-site
git init && git add -A && git commit -m "Единый ФПЦ — лендинг"
gh repo create efpc-site --public --source=. --push
gh api repos/OWNER/efpc-site/pages -X POST -f source.branch=main -f source.path="/"
```
**Свой домен ефпц.рф:** создать файл `CNAME` в корне с содержимым `ефпц.рф`
(или его punycode) и добавить DNS-запись на `OWNER.github.io`.

**Vercel:** `vercel --yes` (если нужны серверные функции).
